# 01-product-requirements.md

# Product Name

LoanBook

# Product Summary

LoanBook is a personal loan tracking application.

A user can track:

* Loans they gave to others
* Loans they took from others
* Repayments
* Outstanding balances
* Interest
* Expenses
* Reports

This is NOT a lending platform.

This is NOT a bank LMS.

This is NOT a marketplace.

Only one user exists.

No borrower login.

No lender login.

No payment gateway.

No cloud sync.

All data stored locally.

# Core User Flows

Flow 1

Create Given Loan

Flow 2

Create Taken Loan

User → Loans → New Loan → Taken Loan → Save

Flow 3

Record Payment

Flow 4

Close Loan
